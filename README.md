# Student Management System

A simple console-based application built with Java to manage student records while practicing Object-Oriented Programming (OOP) and Java Collections.

---

## Overview

This project is part of my Java learning journey.

The first version focuses on building the core CRUD operations using Java fundamentals without external libraries or databases. My goal is to improve this project step by step as I learn more about backend development.

---

## Features

- Add students
- Show all students
- Search by ID
- Search by name
- Delete by ID
- Delete by name
- Simple console menu

---

## Technologies

- Java 21
- Object-Oriented Programming (OOP)
- Java Collections Framework
- ArrayList
- Scanner
- IntelliJ IDEA
- Git

---

## What I Learned

Through this version, I practiced:

- Creating and using classes
- Encapsulation
- Working with ArrayList
- CRUD operations
- Loops and conditions
- User input handling
- Basic Git workflow

---

## Current Limitations

- Console application only
- Data is stored in memory using `ArrayList`
- Data is lost when the application closes
- No update feature yet
- No database integration
- No input validation
